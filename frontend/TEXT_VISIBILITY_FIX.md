# 文字可見性修復

## 修復的問題

### 1. 文章詳情頁標題可見性 ✅
**問題**: 淺色模式下文章標題使用漸變背景clip導致可見性不佳
**解決方案**: 改為白色文字配合陰影和發光效果

```scss
.article-title {
  color: #ffffff;
  text-shadow: 
    0 2px 4px rgba(0, 0, 0, 0.3),        // 基本陰影
    0 0 20px rgba(25, 118, 210, 0.5),    // 藍色光暈
    0 0 40px rgba(25, 118, 210, 0.3);    // 擴散光暈
}
```

### 2. 文章列表頁描述文字 ✅
**問題**: 深色模式下banner描述文字不夠白
**解決方案**: 確保描述文字在深色模式下為純白色

```scss
.page-subtitle {
  [data-theme="dark"] & {
    color: #ffffff;
    opacity: 1;
    text-shadow: 0 1px 2px rgba(0, 0, 0, 0.3);
  }
}
```

### 3. 文章列表頁標題一致性 ✅
**額外改進**: 同時優化了文章列表頁標題的可見性
**解決方案**: 使用白色文字配合適當的發光效果

```scss
.page-title {
  color: #ffffff;
  text-shadow: 
    0 2px 4px rgba(0, 0, 0, 0.3),
    0 0 20px rgba(255, 255, 255, 0.3);
    
  [data-theme="dark"] & {
    text-shadow: 
      0 2px 4px rgba(0, 0, 0, 0.5),
      0 0 20px rgba(66, 165, 245, 0.5);
  }
}
```

## 視覺效果

### 淺色模式
- **文章標題**: 白色文字 + 藍色光暈效果 = 清晰可見
- **頁面標題**: 白色文字 + 白色光暈效果 = 優雅顯示
- **描述文字**: 半透明白色 = 柔和易讀

### 深色模式  
- **文章標題**: 白色文字 + 強化藍色光暈 = 科技感十足
- **頁面標題**: 白色文字 + 藍色光暈 = 一致的科技風格
- **描述文字**: 純白色 + 陰影 = 完美對比度
