## 🎯 建議開發順序（更新後）

### Phase 1：首頁重構 + SEO 基礎（影響最大 + 低成本順手做）

1. Hero Section — Tagline 升級 + CTA 按鈕
2. 首頁加入精選 Projects 區塊 + 技術棧速覽
3. 首頁底部 Contact CTA
4. Open Graph Tags（順手，幾個 meta tags）
5. Sitemap.xml + robots.txt + Canonical URLs（順手，10 分鐘）
6. 自訂 404 頁面

### Phase 2：核心頁面深化

7. Projects 詳情頁（重點 — 每個專案的完整故事）
8. Projects 篩選 / 分類
9. About 頁面內容升級 + 職涯時間軸
10. Contact 頁面 + 表單

### Phase 3：體驗與品質提升

11. 滾動動畫 + 頁面轉場
12. 圖片最佳化
13. Lighthouse 優化到 90+
14. 無障礙設計
15. Structured Data / JSON-LD

### Phase 4：錦上添花

16. 訪客分析
17. Resume PDF 下載
18. Testimonials（有素材時再加）
19. 服務項目說明
20. CI/CD + Lighthouse CI
